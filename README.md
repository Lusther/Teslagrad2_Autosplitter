# Teslagrad 2 Autosplitter

An [ASL](https://github.com/LiveSplit/LiveSplit.AutoSplitters) autosplitter script for [Teslagrad 2](https://store.steampowered.com/app/1698220/Teslagrad_2/) speedruns, used with [LiveSplit](https://livesplit.org/).

## Features

- Auto-start on new file creation
- Auto-reset on save slot deletion
- Splits for all **10 skills**: Blink, Blue Cloak, Water Blink, Mjolnir, Power Slide, Axe, Blink Wire Axe, Red Cloak, OmniBlink, Double Jump
- Splits for all **7 bosses**: Hulder, Moose, Fafnir, Halvtann, Galvan, Troll, Elenor
- Optional scroll collection splits

## Installation

1. Open [LiveSplit](https://livesplit.org/)
2. Right-click > Edit Splits
3. Set the game name to **Teslagrad 2**
4. The autosplitter will be available automatically — click **Activate**
